# MissedCall Reply

A free, open-source Android app that automatically sends an SMS 
to anyone who missed-calls you.

## Features
- Auto SMS reply for all missed calls
- Time window mode (e.g. only reply between 10 AM – 3 PM)
- Custom reply message
- Runs in background, auto starts after reboot
- No ads, no account, no internet required

## Permissions
- READ_PHONE_STATE — detect incoming calls
- READ_CALL_LOG — detect missed calls  
- SEND_SMS — send auto reply
- RECEIVE_BOOT_COMPLETED — auto start on reboot
